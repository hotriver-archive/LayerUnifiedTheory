## 📘 Foreword（前書き）

このリポジトリは、専門家ではない一個人が
**「宇宙はこういう構造かもしれない」** という思考実験としてまとめた
概念モデル（レイヤー統一理論/Layer Unified Theory: LUT）の保管場所です。

重力・宇宙膨張・銀河構造・ブラックホールといった複数の宇宙論的テーマを、共通の仮定から再解釈する試みを行っています。

> **Note**
> 数学・物理学の正式な教育を受けておらず、数式の厳密な証明やテンソル計算などは扱えません。
> そのため本内容は、“完全な理論”ではなく **アイデアの骨格だけを提示した未完成のメモ** です。

## 💬 Contributions are Welcome

もし専門的知識をお持ちの方で、
ここにある概念を数学的に補強したり、より厳密な形式に変換できる方がいれば、
**Issue** や **Pull Request** を通じて自由に提案していただければ幸いです。

このリポジトリの目的は、
**「思いつきを腐らせず、誰かの手に渡る可能性を残すこと」** です。
